## Critical gotcha

`useListSubUnits`, `getListSubUnitsQueryKey`, `useListEnergySources`, `getListEnergySourcesQueryKey` are **NOT exported** from `@workspace/api-client-react` — they were not generated by Orval. Do not import them.

**Why:** Orval generates hooks only for paths defined in `openapi.yaml`. These endpoints exist in code but are not in the OpenAPI spec.

**How to apply:** For new pages that need sub-units or energy sources, use direct `axios` calls with the `authHeader()` helper or add them to the OpenAPI spec first.
